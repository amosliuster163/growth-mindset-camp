# 第166期 - 选择过载3.0：为什么选择越多，你越痛苦

> **核心概念**：选择过载（Choice Overload）进阶
> **一句话**：选择多不是好事，它让你更不快乐
> **完成时间**：约5分钟

---

## 📖 故事开场

你去超市买果酱。

2种果酱→你很快选了一种，很满意。

24种果酱→你选了15分钟，最后还是不满意。

为什么？因为**选择越多，你的期望越高，后悔的可能性越大。**

---

## 🧠 核心模型

### 选择过载的四个后果

| 后果 | 说明 |
|------|------|
| **决策瘫痪** | 选项太多，干脆不选 |
| **机会成本焦虑** | "我选了这个，就错过了其他" |
| **期望升高** | "24种里选一个，必须完美" |
| **后悔增加** | "也许另一个更好" |

**更多选择 ≠ 更快乐。有时恰恰相反。**

---

## 🔧 实战：简化选择

### 三原则

**1. 主动限制选项**
- 买东西前先设标准："我只看X品牌/价位"
- 约会时先列核心条件："必须有X，其他无所谓"

**2. 设定"足够好"标准**
- 不追求"最佳"，追求"够用"
- "满意就好"比"完美"更实际

**3. 减少日常选择**
- 固定早餐
- 固定穿搭风格
- 把选择额度留给重要的事
