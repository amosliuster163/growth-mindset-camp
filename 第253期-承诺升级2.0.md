# 第253期：承诺升级2.0

> "你已经在这条错误的路上走了这么远，还要继续走下去吗？"

---

## 💡 概念

承诺升级（Escalation of Commitment）：
**明明知道错了，却因为已经投入太多而继续错下去。**

2.0版本不只是"及时止损"，而是：
1. 理解为什么人会上"沉没成本"的当
2. 学会区分"坚持"和"死磕"
3. 建立"退出规则"，在投入前就定义好什么时候该停

---

## 📖 故事

老张花5万块开了一家奶茶店。

3个月后，每天亏500块。

朋友说：关了吧，别亏了。

老张说：已经投了5万了，关了不就全没了？再撑撑。

又撑了6个月，又亏了9万。

最后关了店，亏了14万。

如果3个月时就关，只亏5万。

那多出来的9万，不是生意亏的，是"不甘心"亏的。

## 🧠 模型

### 承诺升级3陷阱

**1. 沉没成本陷阱**
- "已经花了这么多钱/时间，不能放弃"
- 真相：已经花的已经没了，不是继续投入的理由
- 正确问题：如果今天重新开始，我还会做这个决定吗？

**2. 面子陷阱**
- "放弃了别人会怎么看我"
- 真相：别人没那么关注你
- 正确问题：面子重要还是结果重要？

**3. 希望陷阱**
- "再试一次说不定就好了"
- 真相：没有数据支撑的"希望"只是幻想
- 正确问题：有什么证据表明下一次会不同？

### 退出规则模板

在开始任何投入前，先写好：

```
如果我投入了X（时间/金钱/精力）后，
仍然没有看到Y（具体指标），
我就会退出/调整方向。
```

例如：
- 如果投入3个月后日活不到100，就换方向
- 如果亏了本金的30%，就止损
- 如果学了6个月还是不喜欢，就换技能

### 坚持 vs 死磕

| 坚持 | 死磕 |
|------|------|
| 有数据证明在进步 | 没有变化或更差了 |
| 有明确的里程碑 | 没有阶段性目标 |
| 定期复盘和调整 | 一成不变地做同样的事 |
| 知道什么时候该调整 | 不知道什么时候该停 |

## ✅ 今日任务

1. 想一件你正在投入但效果不好的事
2. 问自己：如果今天是第一天，我还会开始做这件事吗？
3. 如果答案是否定的，写下你的退出规则
