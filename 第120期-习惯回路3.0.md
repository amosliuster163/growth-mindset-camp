你想改变，告诉自己"明天一定不刷手机"。

但第二天，一模一样的事又发生了。

你不是没有意志力。你是**没有设计好你的习惯系统**。

## 🧠 核心概念

### 什么是习惯回路？

查尔斯·杜希格在《习惯的力量》中提出：**每个习惯由三个部分组成**

```
触发 → 行为 → 奖励
```

- **触发**：什么启动了习惯（闹钟响了 → 摸手机）
- **行为**：习惯动作本身（刷社交媒体）
- **奖励**：为什么大脑会重复（多巴胺、信息新鲜感）

**你想改变习惯，必须改变回路中的至少一个环节。**

### 习惯回路的3.0升级

**1.0：意识到习惯的存在**
- "我每天刷手机30分钟"

**2.0：理解习惯回路**
- "闹钟是触发，刷手机是行为，多巴胺是奖励"

**3.0：重新设计回路**
- "把触发改掉：闹钟放远一点"
- "把行为替换：拿起手机前先做10个俯卧撑"
- "把奖励升级：运动后的内啡肽比多巴胺更持久"

## 💡 思维模型

### 习惯设计框架

**不要靠意志力，要靠系统设计。**

意志力是有限资源，系统是自动运行的。

### 4步习惯设计法

**步骤1：识别现有回路**
- 触发是什么？（时间、地点、情绪、前一个动作）
- 行为是什么？（具体动作）
- 奖励是什么？（身体/心理感受）

**步骤2：重新设计触发**
- 让坏习惯的触发更难出现
- 让好习惯的触发更容易出现
- 例：想少刷手机 → 把手机放卧室外

**步骤3：替换行为**
- 不要"戒掉"，要"替换"
- 同样的触发和奖励，换一个新行为
- 例：压力时想吃零食 → 改为深呼吸10次

**步骤4：放大奖励**
- 让好习惯的奖励更明显
- 例：运动后记录"成就感"，让大脑记住这个感觉

### 习惯叠加公式

**"在做[已有习惯]之后，我会做[新习惯]。"**

- 喝完早咖啡后 → 冥想5分钟
- 晚饭后 → 散步15分钟
- 睡前 → 写3件感恩的事

**把新习惯"挂"在已有习惯上，成功率提高3倍。**

## 🎯 行动任务

### 今日挑战：设计一个习惯回路

花5分钟，完成以下步骤：

1. **选择一个你想培养的习惯：**
   - 例：每天运动、早起、阅读、冥想

2. **用4步框架设计：**
   - 触发：什么时候/什么场景下做？
   - 行为：具体做什么？（越具体越好）
   - 奖励：做完后有什么奖励？
   - 叠加：可以挂在哪个已有习惯上？

3. **写下你的习惯公式：**
   - "在做______之后，我会______。"

4. **试运行7天：**
   - 每天记录：做了吗？感觉如何？
   - 7天后评估：需要调整哪个环节？

### 实用技巧

**两分钟规则**
- 新习惯的前两周，只做"两分钟版本"
- 想跑步？先只跑2分钟
- 想读书？先只读2页
- 重点不是量，是建立回路

**环境设计**
- 想让习惯更容易 → 减少摩擦
- 想让习惯更困难 → 增加摩擦
- 例：想多运动 → 前一天晚上把运动服放床头

**习惯追踪**
- 用日历打勾，连续7天不中断
- 视觉反馈本身就是一种奖励
