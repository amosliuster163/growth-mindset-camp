# 第263期：沉没成本陷阱4.0 - 学会"及时止损"的智慧

## 📖 今日故事

**王安电脑公司的陨落**

1980年代，王安电脑是全球最大的文字处理系统制造商，市值50亿美元，王安本人位列全美第五富豪。

但个人电脑兴起时，王安坚持"不兼容"路线——他拒绝让自家电脑与IBM兼容，因为**已经投入了太多资源在 proprietary 技术上**。

"我们已经在这条路上走了太远，不能回头。"

结果：1992年，王安电脑申请破产保护。

**沉没成本**让一个帝国走向了绝路。

## 🧠 思维模型

### 沉没成本陷阱（4.0版）

**定义**：已经发生且不可收回的成本（时间、金钱、精力），不应该影响你未来的决策。

**核心公式**：
```
理性决策 = 只看未来成本和收益
非理性决策 = 被过去的投入绑架
```

### 4个升级认知

**1.0**：知道"沉没成本不是成本"（理论）
**2.0**：能识别自己正在被沉没成本绑架（觉察）
**3.0**：敢于承认错误并及时止损（勇气）
**4.0**：建立"止损机制"，让及时止损成为习惯（系统）

### 生活中的沉没成本

| 场景 | 沉没成本 | 理性选择 |
|------|---------|---------|
| 难看的电影 | 已经买了票 | 不好看就走，别浪费时间 |
| 不合适的专业 | 学了3年 | 转专业还是继续？看未来不看过去 |
| 失败的创业 | 投了100万 | 该停就停，别"再投点试试" |
| 错误的感情 | 在一起5年 | 别因为"都这么久了"而不放手 |

## 💡 实操工具

### "零基思考"法

**步骤**：
1. 假设你现在是从零开始
2. 问自己："如果我没有投入过任何东西，我还会做这个选择吗？"
3. 如果答案是"不会" → 立刻止损
4. 如果答案是"会" → 继续，但要重新规划

### 止损清单

每周问自己3个问题：
- 我正在坚持的事情，是因为它有价值，还是因为我投入了太多？
- 如果今天是我第一天接触这个项目/关系/选择，我还会继续吗？
- 我需要放弃什么，才能拥抱更好的未来？

## 🎯 今日任务

**"零基思考"练习**：

选一件你正在坚持的事情（学习、项目、习惯、关系），用"零基思考"法重新评估：

1. 写下你在这件事上的投入（时间/金钱/精力）
2. 假设一切归零，你还会选择它吗？
3. 如果不会，制定一个"止损计划"
4. 如果会，写下你继续的理由和下一步行动

**输出格式**：
```
我正在坚持的是：______
我的沉没成本是：______
零基思考后，我的选择是：______
我的下一步行动是：______
```

## 💬 思考题

为什么"承认错误"这么难？

沉没成本背后，其实不是钱的问题，是**自尊心的问题**。承认错误 = 承认自己当初的判断是错的。

但真正聪明的人都知道：**及时止损，才是最大的赚。**
