# 第217期 - 光环效应

## 🎯 核心概念

**光环效应**：一个人某方面优秀，你就觉得他方方面面都优秀。

> "长得好看的人，连犯错都显得有魅力。"

---

## 📖 故事

1970年代，心理学家做过一个实验：

给两组学生看同一位老师的讲课录像。唯一区别是，A组看到的是老师亲切友好的片段，B组看到的是老师严厉冷漠的片段。

然后让两组学生评价这位老师的外貌、口音、甚至教学习惯。

结果：A组学生不仅觉得老师教得好，还觉得他**长得更好看、口音更悦耳、小毛病也更可爱**。

同一张脸，同一种声音，因为一个"好"的印象，其他所有方面都被美化了。

这就是光环效应：一个好印象会扩散到你评价的所有维度。

现实中：
- 名校毕业 = 你觉得他什么都懂
- 有钱 = 你觉得他有品位、有见识
- 一个产品做得好 = 你觉得他家所有产品都好

---

## 🧠 思维模型

光环效应的运作方式：

1. **第一印象锚定** → 某个突出特征成为锚点
2. **特征扩散** → 从这个锚点推断其他特征
3. **选择性关注** → 只注意支持光环的证据
4. **忽略矛盾** → 反面证据被弱化处理

对抗方法：

1. **分开评估** —— 每个维度独立打分，别互相影响
2. **延迟判断** —— 不要凭第一印象下结论
3. **找反例** —— 刻意思考"这个光环可能不成立的情况"
4. **数据>感觉** —— 用具体指标代替整体印象
