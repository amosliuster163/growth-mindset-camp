# 第139期 - 时间阻塞4.0：让每一天都高效运转

> **核心概念**：时间阻塞（Time Blocking）进阶版
> **一句话**：不是管理时间，而是管理注意力
> **完成时间**：约5分钟

---

## 📖 故事开场

你有个待办清单：10件事。

一整天忙下来，只做了3件。

不是时间不够，而是**你一直在切换任务**。

回个消息→写段文字→看个邮件→刷个手机→又回个消息……

每次切换，大脑需要5-15分钟重新进入状态。

**你真正工作的时间，可能只有2小时。**

---

## 🧠 核心模型

### 时间阻塞4.0：能量匹配法

不是简单地把时间分块，而是**把任务匹配到你的能量高峰**。

```
高能量时段（早上9-11点）→ 深度工作（创作、编程、决策）
中能量时段（下午2-4点）→ 中等任务（开会、沟通、审校）
低能量时段（下午4-6点）→ 低认知任务（回邮件、整理文件）
```

---

## 🔧 实战：设计你的一天

### 四步法

1. **找到你的能量曲线**（记录一周，看什么时候效率最高）
2. **把最重要的事放在高能量时段**
3. **批量处理同类任务**（集中回消息，而不是随时回）
4. **每90分钟休息10分钟**（大脑的注意力周期约90分钟）

### 示例时间表

```
08:00-08:30 → 晨间规划（列优先级）
08:30-10:00 → 深度工作（不可打扰）
10:00-10:15 → 休息
10:15-12:00 → 深度工作
12:00-13:30 → 午餐+散步
13:30-15:00 → 会议/沟通
15:00-15:15 → 休息
15:15-16:30 → 中等任务
16:30-18:00 → 低认知任务+收尾
```

---

## 💡 关键原则

**1. 一个时段，只做一件事**
多任务处理是效率的敌人。

**2. 保护深度工作时段**
关掉通知，告诉别人你不在。

**3. 计划明天，今天下班前**
这样明天一早就知道该做什么，不浪费时间决定。
