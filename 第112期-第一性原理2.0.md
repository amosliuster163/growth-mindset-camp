# 第112期 - 第一性原理2.0：如何从本质思考问题

> **核心模型**：第一性原理进阶（First Principles 2.0）
> **一句话**：不是"别人怎么做"，而是"这件事的本质是什么"。

---

## 📖 名人故事

埃隆·马斯克，特斯拉和SpaceX CEO。他说：

**"我们用第一性原理思考，而不是类比思考。"**

**类比思考：别人怎么做，我跟着做。**
**第一性原理：这件事的本质是什么，从本质推导解决方案。**

**比如造火箭：**
- **类比思考：NASA怎么做，我跟着做**
- **第一性原理：火箭的本质是什么？材料成本是多少？从本质推导，发现成本可以降低10倍**

**这就是第一性原理：从本质思考，而不是从经验思考。**

## 🧠 核心洞察

**什么是第一性原理2.0？**

第一性原理：从本质思考，而不是从经验思考。

第一性原理2.0：不仅知道第一性原理，还掌握拆解和重构的方法。

**两种思维对比**：

| 思维 | 问题 | 例子 |
|------|------|------|
| 类比思维 | "别人怎么做" | 别人做电动车，我也做 |
| 第一性原理 | "本质是什么" | 交通的本质是移动，电池是能源 |

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 经验主义 | "一直这样做" | 问：为什么这样做？ |
| 从众思维 | "大家都做" | 问：本质是什么？ |
| 过度复杂 | "想得太复杂" | 拆解到最基本 |

---

## ✅ 突破方法

### 1. 拆解问题

| 步骤 | 做法 |
|------|------|
| 1. 写问题 | "我想解决什么？" |
| 2. 拆到本质 | "最基础的事实是什么？" |
| 3. 忽略经验 | "别人怎么做不重要" |
| 4. 重新推导 | "从本质推导解决方案" |

### 2. 问5个为什么

| 问题 | 说明 |
|------|------|
| 为什么这样做？ | 找原因 |
| 为什么必须这样？ | 找假设 |
| 这个假设成立吗？ | 验证假设 |
| 如果不这样呢？ | 找替代 |
| 本质是什么？ | 找核心 |

### 3. 重新构建

| 方法 | 做法 |
|------|------|
| 从0开始 | 假设一切重来 |
| 找基本事实 | 不可再分的事实 |
| 重新推导 | 从基本事实推导 |
