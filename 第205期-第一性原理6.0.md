# 第205期 - 第一性原理6.0

## 🎯 核心概念

**第一性原理**：回到最基础的真相，从头推导，而不是类比。

> "我们习惯用类比思考，但类比会限制创新。"
> — 埃隆·马斯克

---

## 📖 真实案例

### 马斯克造火箭的思维方式

**传统思维（类比法）**：
- 火箭很贵 → SpaceX做不成 → 放弃

**第一性原理思维**：
1. 火箭的材料成本是多少？→ 只占2%
2. 为什么发射这么贵？→ 火箭一次性使用
3. 能不能回收？→ 能！
4. 结果：猎鹰9号回收，发射成本降低90%

---

## 🧠 三问自检法

遇到问题时：

| 问题 | 目的 |
|------|------|
| 这件事最基础的事实是什么？ | 剥离假设 |
| 我是不是在盲目类比？ | 打破惯性 |
| 从头推导会怎样？ | 发现新路径 |
