# 第79期 - 路径依赖3.0：如何跳出历史轨迹

> **核心模型**：路径依赖进阶（Path Dependence 3.0）
> **一句话**：历史决定现在，但你可以决定未来。

---

## 📖 名人故事

QWERTY键盘，1868年设计。

**设计初衷：降低打字速度，防止打字机卡键。**

**150年过去了，打字机早已淘汰，但我们还在用这个降低速度的键盘。**

**这就是路径依赖：过去的选择，锁定现在的选择。**

**但你可以选择Dvorak键盘，选择更高效的布局。**

**路径依赖3.0：知道历史在锁定你，主动跳出。**

---

## 🧠 核心洞察

**什么是路径依赖3.0？**

路径依赖：过去决定现在。

路径依赖3.0：不仅知道历史在锁定你，还主动创造新路径。

**例子**：

| 领域 | 历史路径 | 新路径 |
|------|----------|--------|
| 键盘 | QWERTY | Dvorak |
| 职业 | "一直做这个" | "转行做新事" |
| 投资 | "一直买这只" | "分散投资" |

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 惯性思维 | "一直这样" | 问：还有更好的吗？ |
| 转换成本 | "换起来太麻烦" | 计算长期收益 |
| 身份锁定 | "我就是这种人" | 重新定义自己 |

---

## ✅ 突破方法

### 1. 识别锁定

| 问题 | 说明 |
|------|------|
| 我为什么在做这件事？ | 因为过去投入，还是因为现在有价值？ |
| 如果从零开始，我会选这个吗？ | 验证当前选择 |
| 什么在锁定我？ | 习惯、成本、身份 |

### 2. 计算转换成本

| 成本 | 说明 |
|------|------|
| 时间成本 | 学习新技能的时间 |
| 金钱成本 | 转换的直接成本 |
| 机会成本 | 不转换的长期损失 |

### 3. 创造新路径

| 方法 | 做法 |
|------|------|
| 小步尝试 | 先试水，不全押 |
| 寻找榜样 | 看别人怎么跳出的 |
| 重新定义 | "我不是X，我是Y" |
