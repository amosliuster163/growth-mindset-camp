# 🧠 思维成长训练营 - 第8期：情绪推理

> **主题**：为什么你控制不了情绪？
> **核心模型**：情绪推理 (Emotional Reasoning)
> **金句**：感觉不是事实，情绪不是真相。

---

## 📖 故事：那个被"感觉"骗了一辈子的女人

小美，30岁，职场女性。

她有一个特点：**把感觉当事实**。

开会时领导没看她，她觉得"他对我有意见"。

同事没跟她打招呼，她觉得"他们在孤立我"。

男朋友回消息慢了，她觉得"他不爱我了"。

每次她都因为这些"感觉"做出反应：焦虑、质问、争吵。

后来她去看了心理咨询师。

咨询师说了一句话：**"你的感觉是真的，但感觉背后的解读不一定是真的。"**

小美愣住了。

她第一次意识到：**"我觉得"不等于"事实是"。**

## 🧠 思维模型：情绪推理

### 核心逻辑

情绪推理是一种认知扭曲：

> **"我感觉是这样，所以事实就是这样。"**

常见表现：

1. **焦虑 → "一定有什么坏事要发生"**
2. **内疚 → "我一定做错了什么"**
3. **愤怒 → "他一定是故意的"**
4. **自卑 → "我一定很差劲"**

### 情绪推理的陷阱

| 情绪 | 推理 | 事实可能 |
|------|------|---------|
| 焦虑 | "我肯定搞砸了" | "还没开始，什么都没发生" |
| 愤怒 | "他是针对我" | "他可能只是心情不好" |
| 自卑 | "我不如别人" | "你在拿自己的短处比别人的长处" |
| 绝望 | "一切都完了" | "只是这一次失败了" |

### 适用场景
- 情绪激动时做决定前
- 觉得"被针对"时
- 自我评价过低时
- 任何"感觉很强"的时刻

### 反常识洞察
- **情绪是真实的，但情绪告诉你的故事不一定是真的**
- **大脑在情绪下会编造"合理解释"，这些解释往往是错的**
- **真正的情绪管理不是"压抑情绪"，是"质疑情绪告诉你的故事"**

## 📝 本周行动任务

**任务：情绪日记**

1. **本周每次情绪强烈时（焦虑、愤怒、自卑、绝望），立刻记录**
2. **写下：什么事件触发了情绪？**
3. **写下：你的第一反应是什么？（"他觉得我不行"、"他针对我"……）**
4. **写下：有什么证据证明这个反应是对的？**
5. **写下：有什么证据证明这个反应可能是错的？**
6. **周末回顾：有多少次你的"感觉"被证明是错的？**

**完成标准**：完成至少3次情绪记录，并在评论区分享你的发现
