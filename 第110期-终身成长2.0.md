# 第110期 - 终身成长2.0：如何把成长变成终身习惯

> **核心模型**：终身成长进阶（Lifelong Growth 2.0）
> **一句话**：成长不是一次旅程，而是一生的习惯。

---

## 📖 名人故事

本杰明·富兰克林，美国开国元勋。他从普通印刷工，成为政治家、科学家、作家、外交家。

**他的秘诀：每天问自己两个问题：**
- **早上：今天我要做什么好事？**
- **晚上：我今天做了什么好事？**

**他坚持了50年。**

**这就是终身成长：不是一时努力，而是一生坚持。**

---

## 🧠 核心洞察

**什么是终身成长2.0？**

终身成长：一生坚持成长。

终身成长2.0：不仅知道终身成长，还设计了自动成长的系统。

**成长系统**：

| 组件 | 做法 |
|------|------|
| 输入 | 每天阅读/学习 |
| 处理 | 反思/写笔记 |
| 输出 | 写作/教学/实践 |
| 反馈 | 从结果中学习 |

---

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 阶段性成长 | "学完就停" | 建立持续系统 |
| 无输出 | "只输入不输出" | 教是最好的学 |
| 无反思 | "做了不想" | 定期复盘 |

---

## ✅ 突破方法

### 1. 设计成长系统

| 组件 | 每日行动 |
|------|----------|
| 输入 | 读10页书 |
| 处理 | 写3条笔记 |
| 输出 | 写200字/教1个人 |
| 反馈 | 记录1个教训 |

### 2. 微习惯

| 策略 | 做法 |
|------|------|
| 最小行动 | 每天只做1分钟 |
| 绑定习惯 | 和已有习惯绑定 |
| 追踪进度 | 记录连续天数 |

### 3. 成长档案

| 方法 | 做法 |
|------|------|
| 写成长日记 | 每天记录成长 |
| 定期回顾 | 每月看进步 |
| 庆祝里程碑 | 增强动力 |

---

## 📝 本周行动任务

1. 设计你的成长系统（输入/处理/输出/反馈）
2. 设定每日微习惯（最小行动）
3. 开始记录成长日记
4. 执行7天，回顾效果

**本周目标：用终身成长2.0，把成长变成终身习惯。**
