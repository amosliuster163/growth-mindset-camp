# 第252期：决策疲劳3.0

> "你今天做的每一个决定，都在消耗你明天的决定力。"

---

## 💡 概念

决策疲劳（Decision Fatigue）：
**你的决策力是有限的资源，每做一个决定都在消耗它。**

3.0版本不只是"减少决定"，而是：
1. 理解决策力像肌肉——用多了会累，但可以训练
2. 区分"高价值决策"和"低价值决策"，把精力留给前者
3. 通过"自动化"和"规则化"释放决策带宽

---

## 📖 故事

扎克伯格为什么总是穿同一件灰色T恤？

不是因为没钱买衣服，而是因为他不想在"今天穿什么"这种小事上浪费决策力。

他每天要做数百个重要决定，如果每个小决定都要消耗精力，到了下午就做不出好决定了。

乔布斯也是（黑色高领衫），爱因斯坦也是（同款西装）。

不是他们没品位，而是他们知道：决策力要花在刀刃上。

## 🧠 模型

### 决策带宽管理4策略

**1. 自动化低价值决策**
- 每天穿类似的衣服
- 固定早餐吃什么
- 设定固定的运动时间和方式
- 让"不需要想"的事情自动运行

**2. 建立决策规则**
- "如果价格超过500块，等24小时再决定"
- "如果别人推荐，先问：他自己用了吗？"
- 规则 = 提前做好的决定 = 不消耗决策力

**3. 把好决定放在前面**
- 早上决策力最强 → 做重要决定
- 下午决策力下降 → 做例行工作
- 晚上决策力最弱 → 不做任何决定

**4. 减少选项**
- 超市里30种牙膏 → 选10分钟
- 超市里3种牙膏 → 选10秒
- 选项越多，决策力消耗越大

### 决策疲劳的危险信号

| 信号 | 含义 |
|------|------|
| 开始拖延做决定 | 决策力已经透支 |
| 选择"默认选项" | 大脑在偷懒 |
| 冲动消费 | 决策力耗尽后的本能反应 |
| 对小事也烦躁 | 决策疲劳的情绪表现 |

### 决策力训练

- 冥想：提升专注力和决策质量
- 充足睡眠：决策力在睡眠中恢复
- 运动：提升大脑整体功能
- 合理饮食：血糖稳定 = 决策力稳定
