# 第80期 - 成长复盘：如何从经验中快速学习

> **核心模型**：成长复盘（Growth Review）
> **一句话**：经历不等于经验，复盘才能成长。

---

## 📖 名人故事

本杰明·富兰克林，美国开国元勋。他有一个习惯：

**每天晚上，他会回顾一天的行为，问自己："我今天做了什么好事？什么可以改进？"**

**他用这个方法，从普通印刷工，成为政治家、科学家、作家。**

**这就是成长复盘：从经验中提取教训，持续改进。**

---

## 🧠 核心洞察

**什么是成长复盘？**

经历≠经验。复盘=从经历中提取经验。

**复盘的3个层次**：

| 层次 | 问题 | 说明 |
|------|------|------|
| 回顾 | "我做了什么？" | 记录事实 |
| 反思 | "为什么这样？" | 分析原因 |
| 改进 | "下次怎么做？" | 设计行动 |

---

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 不复盘 | "做了就做了" | 定期复盘 |
| 只复盘不行动 | "复盘了但没改" | 设计行动 |
| 只关注失败 | "只看哪里错了" | 也关注成功 |

---

## ✅ 突破方法

### 1. 每日复盘

| 问题 | 说明 |
|------|------|
| 我今天做了什么？ | 记录事实 |
| 哪些做得好？ | 肯定自己 |
| 哪些可以改进？ | 找差距 |
| 明天怎么做？ | 设计行动 |

### 2. 每周复盘

| 问题 | 说明 |
|------|------|
| 这周最大的收获？ | 提取经验 |
| 这周最大的教训？ | 避免再犯 |
| 下周的重点？ | 设定目标 |

### 3. 每月复盘

| 问题 | 说明 |
|------|------|
| 这个月最大的进步？ | 看到成长 |
| 这个月最大的遗憾？ | 找改进点 |
| 下个月的目标？ | 设定方向 |

---

## 📝 本周行动任务

1. 选一个你最近经历的事情
2. 问：我做了什么？为什么？下次怎么做？
3. 写下复盘结果
4. 设计一个改进行动

**本周目标：用成长复盘，从经验中快速学习。**
