# 小窗Windows兼容性测试记录

## 测试1：中文文件创建（2026-04-15）

### 任务
在Windows上创建成长思维训练营内容文件（中文UTF-8编码）

### 结果
✅ 文件创建成功，内容完全一致

### 发现问题
⚠️ Windows命令提示符默认用GBK编码读取UTF-8文件，导致控制台显示乱码
- 实际文件内容正确（UTF-8编码）
- 只是`type`命令显示问题

### 解决方案
- 使用PowerShell的`Get-Content`替代`type`
- 或先执行`chcp 65001`切换UTF-8代码页

### 经验教训
以后Windows任务涉及中文文件时，优先用PowerShell命令，或提醒小窗切换代码页。
