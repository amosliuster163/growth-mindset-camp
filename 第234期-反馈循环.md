# 第234期 - 反馈循环

## 🎯 核心概念

**反馈循环**：系统的输出会反过来影响输入，形成自我强化的循环。

> "越做越好，不是口号，是机制。"

---

## 📖 故事

你开始跑步。

第一周：跑了2公里，气喘吁吁。
第二周：跑了3公里，感觉好了一点。
第三周：跑了4公里，发现睡眠变好了。
第四周：你主动想去跑步。

**这就是正反馈循环。**

你跑步 → 身体变好 → 心情变好 → 更有动力跑步 → 跑得更远 → 身体更好...

循环一旦启动，就会自我强化，你不再需要意志力去坚持。

**负反馈循环同样存在：**

你熬夜 → 第二天疲惫 → 不想运动 → 身体变差 → 更容易疲惫 → 继续熬夜...

**关键在于启动第一个正循环，打断第一个负循环。**

---

## 🧠 思维模型

### 如何建立正反馈循环？

1. **找到最小可行行动** —— 小到不可能失败（比如每天做1个俯卧撑）
2. **设计可见的反馈** —— 记录进度，让自己看到变化
3. **庆祝小胜利** —— 每次完成，给自己一个正向奖励
4. **逐步加码** —— 当你习惯后，稍微增加难度

### 常见的反馈循环

- 运动：运动 → 精力提升 → 效率提高 → 更多时间运动
- 学习：学习 → 能力提升 → 获得认可 → 更想学习
- 阅读：阅读 → 思考变深 → 理解更快 → 读得更多
- 社交：主动连接 → 关系加深 → 获得更多支持 → 更愿意社交

## ⚡ 行动任务

今天启动一个正反馈循环：

**选一件小事，做完后在纸上画一个✅。**

明天再做一次，再画一个。连续画7个，你会发现自己停不下来。

**反馈循环的秘密：启动最难，一旦转动起来，就停不下来。**

意志力是有限的，但正反馈循环是无限的。

**不要靠坚持，要靠机制。让系统替你自律。**
