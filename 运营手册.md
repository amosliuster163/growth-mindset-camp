## 📊 项目概览

| 项目 | 内容 |
|------|------|
| **定位** | 30天思维升级训练营，每周一个思维模型 |
| **目标用户** | 25-40岁，有成长意愿但缺乏系统方法的职场人 |
| **核心差异化** | 不是讲知识，是设计体验让用户自己"顿悟" |
| **内容形式** | 故事(20%) + 模型(30%) + 行动任务(30%) + 讨论(20%) |
| **定价建议** | 99元/期（4期），或 299元/全年（12期） |

## 🔄 每周运营SOP

### 周一：发布新内容
- [ ] 上午10:00 发布本期内容
- [ ] 发布渠道：公众号 + 知识星球 + 社群
- [ ] 配引流话术："扫码加入训练营，每周一个思维模型"

### 周二：检查打卡
- [ ] 自动提醒：检查昨日打卡情况
- [ ] 未打卡用户：发送私信提醒

### 周三：进度提醒
- [ ] 发送周中提醒："本周任务还有4天截止"
- [ ] 分享1-2个优质打卡案例

### 周五：精选打卡
- [ ] 精选3-5个优质打卡
- [ ] 制作"本周精选"海报/文章

### 周日：本周总结
- [ ] 发布本周总结 + 下周预告
- [ ] 统计本周打卡率、互动率
- [ ] 收集用户反馈

## 📊 关键指标

| 指标 | 目标值 | 计算方式 |
|------|--------|---------|
| **打卡率** | ≥ 60% | 打卡人数 / 总人数 |
| **互动率** | ≥ 30% | 评论人数 / 总人数 |
| **完课率** | ≥ 50% | 完成4周任务人数 / 总人数 |
| **续费率** | ≥ 40% | 续订下一期人数 / 到期人数 |
| **NPS** | ≥ 50 | (推荐者% - 贬损者%) |
