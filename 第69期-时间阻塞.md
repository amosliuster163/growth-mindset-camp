# 第69期 - 时间阻塞：把时间变成你的资产

> **核心模型**：时间阻塞（Time Blocking）
> **一句话**：不是"有时间再做"，而是"给每件事分配时间"。

---

## 📖 名人故事

卡尔·纽波特，《深度工作》作者。他有一个习惯：

**"我把每天的时间分成块，每块时间只做一件事。"**

**他不看邮件，除非在"邮件时间块"。他不回消息，除非在"沟通时间块"。**

**结果：他用更少的时间，做了更多的事。**

**时间阻塞：把时间变成你的资产，而不是被时间追着跑。**

---

## 🧠 核心洞察

**什么是时间阻塞？**

把一天的时间分成多个块，每块时间只做一类事。

**例子**：

| 时间块 | 任务 | 时长 |
|--------|------|------|
| 8:00-10:00 | 深度工作 | 2小时 |
| 10:00-10:30 | 邮件/消息 | 30分钟 |
| 10:30-12:00 | 会议 | 1.5小时 |
| 14:00-16:00 | 创作 | 2小时 |
| 16:00-16:30 | 运动 | 30分钟 |
| 17:00-18:00 | 学习 | 1小时 |

---

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 过度计划 | "每分钟都排满" | 留缓冲时间 |
| 不执行 | "计划了但不做" | 从1个时间块开始 |
| 被打断 | "总是被打断" | 设置免打扰 |

---

## ✅ 突破方法

### 1. 识别时间类型

| 类型 | 说明 |
|------|------|
| 深度工作 | 需要专注的任务 |
| 浅层工作 | 邮件、消息 |
| 会议 | 团队沟通 |
| 学习 | 自我提升 |
| 休息 | 恢复精力 |

### 2. 设计时间块

| 原则 | 做法 |
|------|------|
| 同类合并 | 邮件集中处理 |
| 深度优先 | 早上做深度工作 |
| 留缓冲 | 每块之间留15分钟 |

### 3. 保护时间块

| 方法 | 做法 |
|------|------|
| 免打扰 | 关闭通知 |
| 告知团队 | 让大家知道你的时间块 |
| 坚持 | 不轻易改变 |
