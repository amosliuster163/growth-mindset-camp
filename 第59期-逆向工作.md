# 第59期 - 逆向工作：从目标倒推行动

> **核心模型**：逆向工作（Working Backwards）
> **一句话**：不是从现状往前推，而是从目标往后推。

---

## 📖 名人故事

杰夫·贝佐斯，亚马逊创始人。他有一个独特的工作方法：

**"从新闻稿开始。"**

在开发新产品前，先写一份未来的新闻稿：

**"今天，亚马逊发布了XXX，它解决了XXX问题，用户反馈XXX。"**

然后问：**"要达到这个新闻稿，我们需要做什么？"**

**这就是逆向工作：从最终结果倒推，而不是从现状正推。**

---

## 🧠 核心洞察

**什么是逆向工作？**

正向工作：现状 → 步骤 → 目标
逆向工作：目标 → 步骤 → 现状

**例子**：

| 目标 | 逆向步骤 |
|------|----------|
| 1年后月入5万 | 10个月：4万 → 6个月：2万 → 3个月：1万 |
| 写一本书 | 出版 → 完稿 → 每周写1万字 → 今天写1000字 |
| 创业成功 | 盈利 → 1000用户 → 100用户 → 第一个用户 |

---

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|
| 目标不清晰 | "我想变好" | 具体化：什么好？多少？ |
| 步骤不现实 | "一个月赚100万" | 验证可行性 |
| 不执行 | "计划完美但不做" | 从第一步开始 |

---

## ✅ 突破方法

### 1. 写新闻稿

| 步骤 | 做法 |
|------|------|
| 1 | 写一份未来的新闻稿 |
| 2 | 描述成功后的状态 |
| 3 | 包括具体数据和反馈 |
| 4 | 问：要达到这个，需要做什么？ |

### 2. 倒推时间线

| 时间 | 里程碑 |
|------|--------|
| 目标日期 | 最终结果 |
| -3个月 | 倒数第二个里程碑 |
| -6个月 | 中间里程碑 |
| -12个月 | 第一个里程碑 |
| 今天 | 第一步行动 |

### 3. 验证可行性

| 问题 | 说明 |
|------|------|
| 这现实吗？ | 验证目标 |
| 有资源吗？ | 验证能力 |
| 有时间吗？ | 验证周期 |

4. 执行第一步

**本周目标：用逆向工作，找到正确的路径。**
