# 第202期 - 系统思维2.0

## 🎯 核心概念

**系统思维**：不只看单个事件，而是看到事件背后的模式和结构。

> "头痛不一定是头的问题，脚痛不一定是脚的问题。"

---

## 📖 真实案例

### 减肥失败的真正原因

| 表面问题 | 深层原因 |
|---------|---------|
| 管不住嘴 | 情绪性进食 |
| 坚持不了运动 | 目标太大，缺乏正反馈 |
| 体重反复 | 生活习惯未改变 |

**系统解决方案**：
1. 找到情绪来源
2. 从每天5分钟运动开始
3. 改变环境（不买零食）

---

## 🧠 三问自检法

遇到问题时：

| 问题 | 目的 |
|------|------|
| 这是症状还是根因？ | 定位真正问题 |
| 什么模式在重复？ | 发现规律 |
| 改变什么最有效？ | 找到杠杆点 |
