# 第272期 - 锚定效应4.0：如何不被"第一印象"绑架

**概念**：锚定效应（Anchoring Effect）  
**版本**：4.0（决策去锚进阶）  
**日期**：2026-04-21

---

## 📖 概念

**锚定效应**：人们在做判断时，会过度依赖接收到的第一个信息（锚点），即使这个信息与决策无关。

Tversky和Kahneman的经典实验：
- 让两组人估计非洲国家在联合国的占比
- 第一组先看一个转盘转到65，然后估计
- 第二组先看一个转盘转到10，然后估计
- 第一组平均估计45%，第二组平均估计25%

**转盘是随机的，与问题毫无关系，但仍然影响了判断。**

## 🎬 故事：乔布斯的定价锚

iPhone 发布时，乔布斯在台上说：

"这款手机，售价……$999。"

观众倒吸一口凉气。

停顿了5秒后，他说：

"开玩笑的。实际售价……$499。"

全场起立鼓掌。

**$999是锚点。** 如果没有它，$499听起来很贵。有了它，$499听起来像白送。

## 🧠 锚定效应的4种表现

### 1. 价格锚定
- 原价¥999，现价¥199 → 你觉得超值
- 但如果直接标价¥199 → 你觉得一般
- **原价可能从来没人按那个价格卖过**

### 2. 薪资锚定
- HR问"你期望薪资多少" → 你先说的数字会成为锚点
- 你说8K，对方可能压到7K
- 你说12K，对方可能谈到10K
- **先开口的定锚，后开口的被锚**

### 3. 目标锚定
- 你第一次跑步跑了3公里 → 这个成为你的锚
- 下次跑2公里就觉得"退步了"
- 但其实2公里也很好
- **第一次的设定可能太低了**

### 4. 印象锚定
- 第一次见面你觉得某人不友好 → 以后看他都不友好
- 第一次合作你觉得某公司不靠谱 → 以后都不考虑
- **第一印象可能是错的，但你很难改变**

---

## 💡 去锚4步法

### 第1步：识别锚点
问自己："我现在依赖的这个数字/印象，是哪里来的？"

### 第2步：寻找反锚
- 对方开价¥999 → 你去查同类产品的真实价格
- HR问期望薪资 → 你先去查市场行情
- 第一次印象不好 → 给自己3次机会重新判断

### 第3步：主动设锚
- 谈判时先开口（你定锚）
- 自我介绍时先展示最强的优势（你定锚）
- 报价时先说最高价（你定锚）

### 第4步：定期重置锚
- 每个月复盘一次你的目标
- 问自己："这个目标是不是被过去的锚限制了？"
- 如果是，大胆提高它
