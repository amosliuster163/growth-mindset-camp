# 第201期 - 复利思维2.0

## 🎯 核心概念

**复利思维**：不仅仅是金钱的复利，更是知识、习惯、关系的复利。

> "每天进步1%，一年后你会强大37倍。"
> — (1.01)^365 = 37.78

---

## 📖 真实案例

### 曾国藩的"日课"

曾国藩每天写日记、读书、反思，坚持数十年。

**他的日课清单**：
- 早起不赖床
- 每日读书十页
- 写日记反思
- 饭后散步千步
- 睡前静坐十分钟

**复利效果**：
- 1年 → 养成好习惯
- 10年 → 学识渊博
- 30年 → 成为一代名臣

---

## 🧠 三问自检法

每次想放弃时，问自己：

| 问题 | 答案 |
|------|------|
| 今天比昨天好吗？ | 是/否 |
| 这个习惯坚持多久了？ | ___天 |
| 放弃的代价是什么？ | ___ |
