# 第100期 - 元认知3.0：如何思考自己的思考

> **核心模型**：元认知进阶（Metacognition 3.0）
> **一句话**：知道自己知道什么，也知道自己不知道什么。

---

## 📖 名人故事

苏格拉底，古希腊哲学家。他说：

**"我唯一知道的，就是我一无所知。"**

**这不是谦虚，是元认知：知道自己不知道。**

**他说："未经审视的人生不值得过。"**

**未经审视的思考，不是真正的思考。**

**这就是元认知：思考自己的思考。**

## 🧠 核心洞察

**什么是元认知3.0？**

元认知：思考自己的思考。

元认知3.0：不仅知道元认知，还系统化监控自己的认知过程。

**3层元认知**：

| 层次 | 问题 | 说明 |
|------|------|------|
| 元认知知识 | "我知道什么？" | 了解自己的知识 |
| 元认知监控 | "我在怎么思考？" | 监控思考过程 |
| 元认知调节 | "我该怎么调整？" | 调整思考策略 |

## ⚠️ 常见陷阱

| 陷阱 | 表现 | 破解 |
|------|------|------|

| 过度自信 | "我什么都知道" | 承认无知 |
| 自我欺骗 | "我以为我知道" | 检验真实水平 |

## ✅ 突破方法

### 1. 元认知知识

| 问题 | 说明 |
|------|------|
| 我真正掌握什么？ | 诚实评估 |
| 我不知道什么？ | 承认无知 |
| 我怎么知道的？ | 评估知识来源 |

### 2. 元认知监控

| 方法 | 做法 |
|------|------|
| 思考日志 | 记录思考过程 |
| 自我提问 | "我在怎么想？" |
| 暂停反思 | 做事中途暂停 |

### 3. 元认知调节

| 策略 | 做法 |
|------|------|
| 调整策略 | 根据效果调整 |
| 寻求帮助 | 承认不足 |
| 持续学习 | 填补知识空白 |
