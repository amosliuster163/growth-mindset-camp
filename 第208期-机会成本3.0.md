# 第208期 - 机会成本3.0

## 🎯 核心概念

**机会成本**：选择A，意味着放弃了B。真正的成本是你放弃的那个选项。

> "你最大的成本，是你正在浪费时间。"

---

## 📖 真实案例

### 老刘的时间账本

**场景**：周六晚上8点，有3小时空闲。

| 选择 | 表面成本 | 机会成本 |
|------|---------|---------|
| 刷短视频 | 免费 | 3小时成长时间 |
| 看一本书 | ¥50 | 无（长期回报） |
| 写一篇公众号 | 3小时 | 但可能带来1000阅读量 |
| 陪孩子做手工 | 0元 | 但建立了亲子关系 |

**最优解**：陪孩子1小时 + 写作1小时 + 阅读1小时

---

## 🧠 三问自检法

做选择前：

| 问题 | 目的 |
|------|------|
| 我放弃的是什么？ | 识别机会成本 |
| 这个时间能创造的最大价值是什么？ | 寻找最优解 |
| 能组合多个选项吗？ | 突破二元思维 |
